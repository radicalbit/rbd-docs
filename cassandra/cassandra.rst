=========
Cassandra
=========

|fdd| includes Apache Cassandra |cassandra-version|.

Where not else stated, follow the `official documentation <http://docs.datastax.com/en/cassandra/2.2/cassandra/cassandraAbout.html/>`_.
Please note that Apache Cassandra differs from Datastax Enterprise Cassandra. When reading Cassandra's documentation, always check whether it is referred
to the open-source Cassandra or to the commercial one provided by Datastax.
