=========
Cassandra
=========

|fdd| includes Apache Cassandra |cassandra-version|.

Where not else stated, follow the `official documentation <http://docs.datastax.com/en/cassandra/2.2/cassandra/cassandraAbout.html/>`_.
Please note that Apache Cassandra differs from DataStax Enterprise Cassandra (DSE). When reading Cassandra's documentation, always check whether it is referred
to the open-source Cassandra (even if hosted in DataStax site) or to the commercial one provided by the aforementioned firm.
