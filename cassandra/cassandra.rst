=========
Cassandra
=========

|fdd| includes Apache Cassandra |cassandra-version|.

Where not else stated, follow the `official documentation <http://docs.datastax.com/en/cassandra/2.2/cassandra/cassandraAbout.html/>`_.
Pay attention that Apache Cassandra differs from Datastax Enterprise Cassandra. When reading documentation about Cassandra, always check if it is referred
to the opensource Cassandra and not the commercial one provided by Datastax.
