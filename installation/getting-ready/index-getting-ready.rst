=============
Getting Ready
=============

This section describes the information and materials you should get ready to install a |rbd-stack| cluster using Ambari. Ambari provides an end-to-end management and monitoring solution for your |rbd-stack| cluster. Using the Ambari Web UI and REST APIs, you can deploy, operate, manage configuration changes, and monitor services for all nodes in your cluster from a central point.
